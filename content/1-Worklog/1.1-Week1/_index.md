### Week 1 Achievements:

* **Understood and mastered core AWS service groups:**
    * **Compute:** Acquired skills to launch and manage virtual servers via EC2.
    * **Storage:** Understood EBS volumes and their attachment to compute instances.
    * **Database:** Successfully initialized and managed relational databases with RDS/Aurora.
    * **Serverless:** Explored event-driven architecture by deploying code without managing servers using Lambda.
    * **AI/ML:** Gained hands-on experience with Generative AI through Amazon Bedrock.
* **Mastered Cost Optimization and Management:**
    * Successfully configured **AWS Budgets** to set up 80% threshold alerts, preventing "Credit Killers."
    * Developed the habit of **Terminating/Deleting** resources immediately after lab completion to optimize costs.
* **Proficiency in the AWS Management Console:**
    * Became familiar with searching for services and configuring technical parameters (AMI, Instance Types, Security Groups).
    * Successfully handled **Model Access** requests within the Amazon Bedrock console.
* **System Administration Skills:**
    * Created Key Pairs and established secure **SSH connections** to Linux instances.
    * Utilized AWS Blueprints to rapidly deploy functional HTTP-based web applications.
* **Security Mindset:**
    * Implemented the **Principle of Least Privilege** by configuring specific Security Group rules.
    * Understood the importance of protecting Account IDs and access credentials during cloud operations.
